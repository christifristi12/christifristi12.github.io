Portfolio Website

Mijn persoonlijke portfolio waarin ik mijn projecten, vaardigheden en ontwikkeling als software developer presenteer.

Omschrijving

Voor dit project heb ik een persoonlijke portfolio ontwikkeld waarin bezoekers mijn projecten, technische vaardigheden en contactgegevens kunnen bekijken.

Inspiratie

Voor het ontwerp heb ik gekeken naar andermans portofolio en kwam ik opeens op het idee dat ik scheuren wou gaan maken in mijn naam dus heb ik gezocht hoe ik dat kon maken en ik zag toen dat je een gradient kon vast maken aan text en bewegen met keyframes. en toen dacht ik dat is misschien ook cool voor de achtergrond. en toen was ik de project pagina's aan het maken en dacht ik ik verander gewoon de kleuren schema naar ieder project.

Kleurenschema

  • blauw groen geel als achtergrond
  • wit voor tekst en knoppen
  • Zwart voor contrast
  • Wit voor aanvullende elementen

Sitemap

De website bestaat uit een homepage met een overzicht van mijn projecten. Elke projectkaart verwijst naar een aparte pagina met meer informatie, gebruikte technieken en screenshots.
